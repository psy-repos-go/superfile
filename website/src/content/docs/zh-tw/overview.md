# superfile 是什麼？

superfile 是一款現代化的終端機檔案管理器，特別重視使用者介面、功能性與易用性。它以 [Go](https://go.dev/) 和 [Bubble Tea](https://github.com/charmbracelet/bubbletea) 打造，結合吸引人的視覺設計與終端機工具的簡潔特性，為檔案管理帶來清新且容易上手的使用方式。

# 為什麼要打造 superfile？

在建立 superfile 之前，我試過許多終端機檔案管理器，但經常對它們的 UI 設計感到失望。因此，我打造 superfile 時，將提供精緻且使用者友善的介面作為首要目標。

# 我為什麼要使用 superfile？

superfile 外觀俐落且賞心悅目，很適合處理輕量的檔案或目錄任務。雖然它的功能可能不像某些終端機檔案管理器那麼完整，但在易用性與設計上表現出色。如果你需要功能完整的檔案管理器，我會推薦 [Yazi](https://github.com/sxyazi/yazi) 或其他工具。不過，如果你想用乾淨的介面處理直覺明確的任務，superfile 會是很好的選擇。
