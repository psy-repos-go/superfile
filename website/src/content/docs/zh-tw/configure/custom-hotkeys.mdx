你可以輸入下列指令來設定：

[點我了解 HOTKEYS_PATH 在哪裡](/zh-tw/configure/config-file-path#hotkeys)

```bash
$EDITOR HOTKEYS_PATH
```

:::caution
請不要使用 ascii code 會和 superfile 控制鍵衝突的快捷鍵：

- `Ctrl+M` - 與 `Enter` 鍵衝突
- `Ctrl+I` - 與 `Tab` 鍵衝突
- `Ctrl+?`, `Ctrl+[` - 與 `Delete` 和 `Backspace` 鍵衝突
:::

### superfile 預設快捷鍵

:::caution
如果你是 vim 使用者，預設快捷鍵可能會讓你討厭 superfile。
:::

superfile 預設快捷鍵的設計概念：

- 所有會變更檔案的快捷鍵都使用 `ctrl+key`（只要你沒有按 ctrl，你的檔案就永遠是安全的）。
- 不會控制檔案的類別，使用單字首字母作為快捷鍵。
